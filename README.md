Exercise Videos Folder

This folder contains exercise videos used in the app.

Due to GitHub size limitations, videos are not stored in the repository.

Download videos from:
https://drive.google.com/drive/folders/1pneo8FHCjOjVMBMN9dhaQw0M_KrqQzb7

Steps:
1. Download all videos
2. Place them inside:
   frontend/assets/videos/
3. Ensure filenames remain unchanged

Note:
Do not rename files, as the app depends on exact filenames.
